jQuery的serialize()方法通过序列化表单值，创建URL编码文本字符串，我们就可以选择一个或多个表单元素，
也可以直接选择form将其序列化，把序列化的值传给ajax()作为url的参数，轻松使用ajax()提交form表单。
$.ajax({
type: ‘post’,
url: ‘your url’,
data: $(“form”).serialize(),
success: function(data) {
// your code
}
});
表单要进行序列化必须给需要的标签添加name属性

2048(我的第一个js入门项目，更新中)

学了一段时间的js，写过一些简单的项目，但大都是参考网上的代码，几乎没有独立完成过。
决定写2048小游戏，这个游戏的完成给了我很大的编程信心。

先看下效果
https://huilingoo.github.io/2048/

首先思路很重要，玩家按下“上下左右”后，发生了啥事儿：
1、随机出现2或者4的方格；
2、判断该方向上的数字是否相等，相等则相加；

用二维数组来表示，i表示横，j表示竖。初始化每个格子都是零，玩家按键时随机变化一个零，然后根据方向判断。
比如按下“右”键，判断每个横排右边的数字是否为零或者相等，如果是则相加。但是会发现累加上最后一个数会出现
数组越界的问题。解决方法是在最外层加上一层格子，以前是4乘4，现在是6乘6。

写完了，问了师兄的建议，回答是代码太乱了。然后自己百度，知道了原来写代码不只是要实现功能，还有编码规范，遵循
五大原则，代码需要优美，足够健壮，扩展性够强。关于编码规范，累积之后，用另外一个章节。

如何快速搭建我的博客

参照网上教程搭建了一个简单博客，使用github+hexo。

hexo是一个快速简洁高效的博客框架，现在开始我们的操作（windows用户）。

1、确保电脑里安装了git和node.js

2、注册一个github账号，创建一个仓库名为：用户名.github.io

3、电脑任意位置单击右键git bash here,执行 npm install hexo-cli -g，完成后执行npm install hexo –save
之后执行hexo -v，出现版本则成功

4、电脑任意盘里新建个文件夹hexo（你的博客文件，名字随意），进入hexo执行下面操作
hexo init
npm install
hexo g
hexo s
最后一步完成，若控制台提示
INFO Start processing
INFO Hexo is running at http://localhost:4000/ . Press Ctrl+C to stop.
浏览器中打开 http://localhost:4000/

5、编辑_config.yml文件
deploy:
type: git
repository: https://github.com/MrLrf/MrLrf.github.io.git
branch: master
保存

6、执行以下操作
npm install hexo-deployer-git –save
hexo clean
hexo generate
hexo server
hexo deploy
个人博客搭建成功，浏览器中输入https://mrlrf.github.io/

后续操作

1、想要给文章添加样式，可直接在.md文件中添加

2、修改文章后，git bash here执行以下操作：
hexo clean
hexo generate
hexo deploy

3、创建新文章 hexo new “文章名”
