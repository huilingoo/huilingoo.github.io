vue如何上手

在开始一个vue项目之前首先了解什么是webpack。它是当下最热门的前端资源模块化管理和打包工具，它可以将很
多松散的模块按照依赖和规则打包成符合生产环境部署的前端资源，还可以将按需加载的模块进行代码分割，等到实际需
要的时候再异步加载。一些相互依赖的模块文件，会被打包成一个或多个js文件，可以减少HTTP的请求次数。
接下来我们将使用webpack构造前端工程，开始一个vue项目。

1、npm install vue—-构建大型应用时推荐npm安装
2、npm install –global vue-cli—–全局安装vue-cli
3、vue init webpack my-project——创建一个基于webpack的新项目
。。。。
一直回车默认就行

到这儿，我们的第一个vue项目就创建好了，Your application is running here: http://localhost:8080
